# VIM

## Modes

* Insert __i__
* Visual __v__
* Command __esc__

## Save

>:w

## Quit

>:q

## Quit without asking

>:q!

## Save and quit

>:x

## Number of Lines

>:set nu

## Quit Number of Lines

>:set nu!

### y3y

>copy 3 lines

### yw

>copy a word

### u

>Undo

### ctrl + r

>Redo

### gg

>beginning of code

### Shift+g

>end of code

### o

>insert a blank line below current line

### Unblock Vim
