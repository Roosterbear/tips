# INTRODUCCIÓN

- [x] __Instalación__ <br/>

```npm install -g typescript```
```tsc -v```

- [x] Al compilar __Typescript__ se convierte en un __Javascript__ <br/>
- [x] Ahora creamos un archivo script.ts <br/>

- [x] __Compilar__

```tsc script.ts```

- [x] Se genera ahora un archivo __script.js__
